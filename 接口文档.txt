（所有参数json对象封装）一、用户管理（父地址：user/）	1 用户注册		地址: userRegister		说明：			a.密码需要确认，MD5加密后再传输		提交参数			userName:String 用户名			password:String 密码		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功			token:String 令牌 成功后才会返回	2 用户登录		地址：userLogin		说明：			a.密码需MD5加密后再传输		提交参数			userName:String 用户名			password:String 密码		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功			token:String 令牌 成功后才会返回	3 查询用户信息		地址：queryUserInfo		说明：			a.用于查询个人信息，包括积分等信息			b.user是一个json对象		提交参数			token:String  令牌			userName:String 用户名		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功			user:{				userName:String 用户名				nick_name:String 昵称				score:int 当前积分				registerTime:String 注册时间				lastLoginTime:String 上次登录时间				lastLoginIp:String 上次登录ip			}	4 修改用户信息		地址：updateUserInfo		说明：			a.接口可以用于修改密码，也可以用于修改昵称等其他个人信息			b.密码需要MD5加密后提交			c.当字段不需要修改时，不要提交该字段（若有原值提交原值也可）		提交参数			userName:String 用户名（必传）			token:String  令牌（必传）			oldPassword:String 原有密码（用于修改密码）			newPassword:String 新密码（用于修改密码）			nickName:String 昵称（用于修改个人信息）		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功			user:{				userName:String 用户名				nick_name:String 昵称				score:int 当前积分				registerTime:String 注册时间				lastLoginTime:String 上次登录时间				lastLoginIp:String 上次登录ip			}二、任务管理（父地址：task/）	1 查询支持任务类型		地址：queryTaskType		说明：			a.用于进入新建任务页面，能新建哪些任务类型			b.及查询各任务类型的单价等默认值			c.taskTypes是一个json数组			d.taskTypeAttrs 是一个json数组，用于该类型的需要添加的字段			  当fieldConten有值时，fieldRemark不用提示			e.当taskTypeAttr中的mustKey为是时，提交任务时若没有填，应提示填写		提交参数			userName:String 用户名（必传）			token:String  令牌（必传）		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功			taskTypes[				{					name:String 任务类型名称					defPrice:int 默认价格					taskTypeAttrs:[						{							fieldName:String 字段key							fieldCname:String 字段中文名称							fieldRemark:String 字段说明（用于提示）							fieldConten:String 字段默认值							mustKey:int 是否必填值（0否，1是）						}……					]				}……			]	2 提交任务		地址：addTask		说明:			a.总价计算得来，不可更改			b.截止时间也不更改			c.taskAttrs与任务类型中的taskTypeAttrs个数是一致的			d.返回成功后，应提示是否继续创建，若选是重新进入新建任务页面，选否进入任务列表页面		提交参数			userName:String 用户名（必传）			token:String  令牌（必传）			taskType:int 任务类型			totalSum:int 数量			unitPrice:int 单价			markupPrice:int加价			totalPrice:int 总价（=（单价+加价）*数量）			limitSpeed:int 限速（每分钟可以派送多少个）			endTime:String 截止时间（默认一天后）			remark:String 备注			taskAttrs[				{				  fieldName:String 字段key				  fieldConten:String 字段实际值				}……			]		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功					3 我发布的任务		地址：myReleaseTask		说明：			a.所有与时间有关的字段若返回0001-01-01 00:00:00，则显示空字符串		提交参数			page:int 跳转第几页			pageSize:int 每页显示多少			以下为可选参数			state:int 状态（1 待发布，2 已发布（待完成），3 已完成，4 未完成自动下架，5 未完成手动下架）			taskId:String 任务号			taskName:String 任务名称 （支持模糊查询）		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功				total:int 总数			totalPage:int 总页数			tasks:[				{					taskId:String 任务号					taskName:String 任务名称					taskTypeName:String 任务类型					releaseTime:String 发布时间					offShelfTime:String 下架时间					accepted:int 已接受					completed:int 已完成					giveUp:int 已放弃					totalSum:int 总数				}			]	4 任务详情		地址：taskDesc		说明:		提交参数			taskId:String 任务号		返回参数			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功				task:{					taskId:String 任务号					taskName:String 任务名称					taskTypeName:String 任务类型					releaseTime:String 发布时间					offShelfTime:String 下架时间					accepted:int 已接受					completed:int 已完成					giveUp:int 已放弃					totalSum:int 总数					totalPrice:int 总价					markupPrice:int 加价					limitSpeed:int 限速（每分钟可以派送多少个）					picture1:String 图片1地址					picture2:String 图片2地址					state:int 状态（1 待发布，2 已发布（待完成），3 已完成，4 未完成自动下架，5 未完成手动下架）					taskTypeAttrs:[						{							fieldName:String 字段key							fieldCname:String 字段中文名称							fieldConten:String 字段值						}……					]				}	5 任务修改		地址：updateTask		说明：			a.修改成功后返回任务列表		提交参数：			taskId:任务号			可修改参数			taskName:String 任务名称			limitSpeed:int 限速（每分钟可以派送多少个）			markupPrice:int 加价			picture1:String 图片1地址			picture2:String 图片2地址			taskTypeAttrs:[						{							fieldName:String 字段key							fieldConten:String 字段值						}……					]		返回参数：			msg:String 接口返回说明			msgCode:int 返回状态码 0 失败 1成功					6 任务派发情况	7 接受任务	8 放弃任务	9 任务确认	10 我的积分详情
