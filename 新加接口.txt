需要增加的接口 （以下接口都需要token）
	1 绑定邀请码接口
		地址：/api/user/bindInviteCode
		说明：需要在"我的"加个页面提交邀请码
		提交参数：
			inviteCode:Sting 邀请码
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
	2 轮播图接口
		地址：/api/common/carousel
		说明：若openUrl为空字符串或为null 点击图片没有任何效果，否则点击图片需要跳转链接

		提交参数
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
			total:int 轮播图的总数
			carousels[
					{ picture:String 图片地址
					  title:String 说明
					  openUrl:String 跳转地址
					}
				……
			]
	3 提交充值
		地址：/api/pay/addRecharge
		说明 :
			a.money页面上显示为元，提交到后台需要转化成分
		提交参数：
			money:int 充值金额（元）（必填）
			payType:int 支付方式（1微信，2 支付宝）（必填）
			payNum:支付流水号（必填）
			remark:备注
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
    4 充值列表
		地址：/api/pay/queryRecharge
		说明: a.money后台返回是分，页面显示需要转化为元
		提交参数：
			page:int 当前几页
			pageSize:int 每页显示多个
			state:int（1 待处理  2 充值成功 3充值失败）
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
			recharges:[
					{
						money:int 金额
						payType:int 支付方式（1微信，2 支付宝）
						payNum:支付流水号
						state:int（1 待处理  2 充值成功 3充值失败）
						summitTime:String 提交时间
						rechargeTime:String 充值时间
					    remark:备注
					}
					……
			]
	5 提现提交
		地址：/api/pay/addApplyCash
		说明 :
			a.money页面上显示为元，提交到后台需要转化成分
		提交参数：
			money:int 提现金额（元）（必填）
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功		
	6 提现列表
		地址：/api/pay/queryApplyCash
		说明: a.money后台返回是分，页面显示需要转化为元
		提交参数：
			page:int 当前几页
			pageSize:int 每页显示多个
			state:int（1 待处理  2 提现成功 3 提现失败）
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
			applyCashs:[
					{
						money:int 金额
						state:int（1 待处理  2 充值成功 3充值失败）
						summitTime:String 提交时间
						handleTime:String 充值时间
						remark:备注
					}
					……
			]		
	7  我的邀请用户
		  地址：/api/user/queryMyInviteUser
		提交参数：
			page:int 当前几页
			pageSize:int 每页显示多个
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功		
			users:[
				userName:String 用户名
				registerTime:String 注册时间
			]
	8 需要一个调用APP的分享接口，接口待定
		是加载提交任务页面
以下对原有接口修改
	1 用户注册
		地址: api/user/userRegister	
		提交参数增加
			inviteCode:Sting 邀请码（非必填）
	2	查询用户信息
		地址：api/user/queryUserInfo	
		返回user内增加
			releaseScore:long 发布积分
			
	3 我的积分详情（这次修改状态）
		地址：queryScoreCz
		说明：
		提交参数：
			page:int 当前几页
			pageSize:int 每页显示多个
			state:int（ 1发布积分 2 奖励积分）
		返回参数：
			msg:String 接口返回说明
			msgCode:int 返回状态码 0 失败 1成功	
			scoreCzs[
				{
					czTime:String 操作时间
					payPrice:int 支出积分
					inPrice:int 收入积分
					balance:int 余额
					remark:String 备注
					state:int（ 1发布积分 2 奖励积分）
				}
			]
